# 爱彼迎项目

照着coderwhy老师的视频教程写出来的项目，模仿爱彼迎网站

演示地址：

https://mundane.ink/airbnb

### 技术栈：

- react18
- css库：styled-components
- 组件库：material-ui
- 状态管理：redux、reduxjs/toolkit
- 其他：classname

### 重要技术点

### **hooks:**

- useState
- useEffect
- useRef
- useSelector
- useCallback
